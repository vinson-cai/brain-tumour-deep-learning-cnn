# brain-tumour-deep-learning-cnn
Proof of concept of deep learning models being used as a helpful tool in radiology.
Trained several convolutional neural networks (CNNs) of differing architectures and performed hyperparameter tuning.
Best model achieved 90%+ accuracy - was trained on a dataset of about 5000 images.
Dataset sourced from Kaggle: https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset
